## Установка Git

**Git** — это система контроля версий, которая необходима для скачивания проекта с GitHub.

### Для Windows:

### Скачайте Git:
1. Перейдите на официальный сайт Git: [https://git-scm.com/](https://git-scm.com/).
2. Нажмите на кнопку "Download for Windows" (или выберите версию для вашей операционной системы).
3. После загрузки запустите установщик.

### Установите Git:
1. В процессе установки оставьте все настройки по умолчанию, просто нажимайте "Next".
2. На финальном этапе нажмите "Install".
3. После установки откройте командную строку (нажмите Win + R, введите "cmd" и нажмите Enter).
4. Введите команду `git --version`, чтобы проверить, что Git установлен правильно.

### Для Mac:
1. Установите Homebrew (если еще не установлен):
   - Откройте Terminal (нажмите Command + Space и введите "Terminal")
   - Вставьте и выполните команду:
     ```
     /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
     ```

2. Установите Git через Homebrew:
   ```
   brew install git
   ```

3. Проверьте установку:
   ```
   git --version
   ```

## Скачивание проекта

У вас есть два способа скачать проект:

### Способ 1: Через командную строку (рекомендуется)

#### Для Windows:
1. Откройте командную строку (Win + R, введите "cmd" и нажмите Enter)
2. Перейдите в папку "Документы" командой:
   ```
   cd %USERPROFILE%\Documents
   ```
3. Создайте новую папку для проекта и перейдите в неё:
   ```
   mkdir marquiz-project
   cd marquiz-project
   ```
4. Скачайте проект командой:
   ```
   git clone https://github.com/latkn/marquiz-preview-tool.git .
   ```
   (Точка в конце команды означает "скачать в текущую папку")

#### Для Mac:
1. Откройте Terminal (Command + Space, введите "Terminal")
2. Перейдите в папку "Документы":
   ```
   cd ~/Documents
   ```
3. Создайте новую папку для проекта и перейдите в неё:
   ```
   mkdir marquiz-project
   cd marquiz-project
   ```
4. Скачайте проект командой:
   ```
   git clone https://github.com/latkn/marquiz-preview-tool.git .
   ```

### Способ 2: Скачивание вручную
1. Создайте папку на вашем компьютере, где будет храниться проект.
2. Откройте командную строку и перейдите в созданную папку командой:
   ```
   cd путь/к/вашей/папке
   ```
3. Скопируйте проект с GitHub командой:
   ```
   git clone https://github.com/latkn/marquiz-preview-tool.git
   ```

## Установка Visual Studio Code

### Для Windows:

#### Способ 1: Через командную строку (только для Windows)
1. Откройте PowerShell от имени администратора (Win + X, затем выберите "Windows PowerShell (Admin)")
2. Выполните команду:
   ```
   winget install -e --id Microsoft.VisualStudioCode
   ```
3. Дождитесь окончания установки

#### Способ 2: Через сайт
1. Перейдите на официальный сайт VSCode: [https://code.visualstudio.com/](https://code.visualstudio.com/)
2. Нажмите кнопку "Download for Windows" (или выберите вашу операционную систему)
3. Запустите скачанный установщик и следуйте инструкциям

### Для Mac:
1. Скачайте VS Code с официального сайта: [https://code.visualstudio.com/](https://code.visualstudio.com/)
2. Нажмите на кнопку "Download for Mac"
3. После скачивания откройте файл .dmg
4. Перетащите иконку VS Code в папку Applications
5. Для запуска из терминала:
   - Откройте VS Code
   - Нажмите Command + Shift + P
   - Введите "shell command"
   - Выберите "Shell Command: Install 'code' command in PATH"

### Открытие проекта в VS Code

#### Способ 1: Через командную строку (рекомендуется)

##### Для Windows:
1. Перейдите в папку с проектом (если вы не там):
   ```
   cd путь/к/папке/с/проектом
   ```
2. Выполните команду:
   ```
   code .
   ```
   VS Code автоматически откроется с вашим проектом

##### Для Mac:
1. Откройте Terminal
2. Перейдите в папку с проектом:
   ```
   cd ~/Documents/marquiz-project
   ```
3. Выполните команду:
   ```
   code .
   ```

#### Способ 2: Через интерфейс VS Code

##### Для Windows:
1. Запустите VS Code
2. Нажмите File → Open Folder (или Ctrl + K, Ctrl + O)
3. Найдите и выберите папку с проектом
4. Нажмите "Select Folder"

##### Для Mac:
1. Запустите VS Code
2. Нажмите Command + O
3. Найдите и выберите папку с проектом
4. Нажмите "Open"

## Установка расширения Live Server

**Live Server** позволяет запускать локальный сервер для просмотра вашего проекта в браузере.

### Откройте VSCode:
1. Нажмите на значок **"Extensions" (Расширения)** в левой боковой панели (или нажмите `Ctrl+Shift+X`).

### Найдите Live Server:
1. В поиске введите **"Live Server"**.
2. Найдите расширение **"Live Server"** от Ritwick Dey и нажмите **"Install"**.

### Перезапустите VSCode:
1. После установки закройте и снова откройте VSCode.

## Запуск проекта

Теперь всё готово для запуска проекта!

### Для Windows:
1. Откройте папку проекта в VSCode:
   Нажмите `File > Open Folder...` и выберите папку `marquiz-preview-tool`, которую вы клонировали ранее.

2. Запустите Live Server:
   В панели **Explorer** (слева) найдите файл `index.html`.
   Щелкните правой кнопкой мыши на файле `index.html` и выберите **"Open with Live Server"**.

3. Просмотрите проект:
   Ваш браузер автоматически откроет проект по адресу:
   [http://127.0.0.1:5500/](http://127.0.0.1:5500/).

### Для Mac:
1. В VS Code найдите файл index.html в списке файлов слева
2. Щелкните правой кнопкой мыши (или Control + клик) на файле index.html
3. Выберите "Open with Live Server"
4. Проект автоматически откроется в вашем браузере по адресу [http://127.0.0.1:5500/](http://127.0.0.1:5500/)

## Возможные проблемы и их решение

### Ошибка при клонировании репозитория:
- **Убедитесь**, что Git установлен правильно.
- Проверьте, есть ли у вас доступ к интернету.

### Live Server не работает:
- **Убедитесь**, что расширение Live Server установлено.
- Проверьте, открыт ли файл `index.html` в VSCode.

### Проект не отображается в браузере:
- **Убедитесь**, что вы запустили Live Server именно на файле `index.html`.

### Специфичные проблемы для Mac:

#### Ошибка "Permission denied" при установке:
- Используйте sudo перед командами установки:
  ```
  sudo brew install git
  ```

#### Homebrew не установлен:
- Проверьте установку командой:
  ```
  brew --version
  ```
- Если не установлен, установите по инструкции выше

#### Terminal не видит команду code:
- Переустановите команду code в PATH через VS Code:
  1. Откройте VS Code
  2. Command + Shift + P
  3. Введите "shell command"
  4. Выберите "Shell Command: Install 'code' command in PATH"
